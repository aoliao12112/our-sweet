做好了，快来夸我

🎉 传阅小板报：全网页端盲改发帖指南！

教一下大家怎么发帖子：

  • 1. 登陆 GitHub
  • 2. 找到一篇之前的 post，比如我这篇，名字叫 Zwx_firstpost 的 html 源码复制一下
  • 3. 点击 Create new file，按照格式命名：xx年/x月/x日/名字/index.html
  • 4. 粘贴刚刚复制的 html
  • 5. 把标题、作者、和这里的正文修改即可！

以后大家都可以直接在网页端整活啦，快来试试看！✨
